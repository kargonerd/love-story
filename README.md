# Love Story

一个用图片、文字与音乐讲述共同回忆的静态故事页面，基于 Flowtime.js 制作。

在线访问：[love.jojokanbao.cn](https://love.jojokanbao.cn/)

## 本地预览

项目不需要构建，使用任意静态文件服务器即可：

```bash
python -m http.server 8000
```

然后访问 <http://localhost:8000>。

## 操作方式

- 按空格键开始或暂停播放
- 右上角按钮控制背景音乐
